# Leap 快速上手

这份说明对应当前配置文件 [lua/plugins/leap.lua](/root/.config/nvim/lua/plugins/leap.lua) 里的映射。

## 当前键位

| 键位 | 模式 | 作用 |
|------|------|------|
| `s` | `n` `x` `o` | 向前跳到两字符目标 |
| `S` | `n` `x` `o` | 向后跳到两字符目标 |
| `gs` | `n` `x` `o` | 跳到其他窗口中的目标 |
| `x` | `x` `o` | 向前 `t` 风格跳转 |
| `X` | `x` `o` | 向后 `T` 风格跳转 |

说明：

- `n` = Normal 模式
- `x` = Visual 模式
- `o` = Operator-pending 模式，比如 `d`、`y`、`c` 后等待目标时

## 最常用的按法

### 1. 向前跳

```text
sab
```

含义：向前找最近的 `ab`。

### 2. 向后跳

```text
Sab
```

含义：向后找最近的 `ab`。

### 3. 跨窗口跳

```text
gsab
```

含义：在其他窗口里找 `ab` 并跳过去。

## 交互规则

按下 `s` 或 `S` 后：

1. 先输入两个字符，比如 `ab`
2. 屏幕上会出现候选目标
3. 如果最近目标足够明确，Leap 会直接跳过去
4. 如果目标很多，会显示标签字母
5. 再按对应标签字母，跳到目标位置

## 和编辑命令配合

因为 `Leap` 也绑定到了 `o` 模式，所以它可以直接作为动作范围。

### 删除到某处

```text
dsab
```

含义：删除到下一个 `ab`。

### 复制到某处

```text
ysab
```

含义：复制到下一个 `ab`。

### 修改到某处

```text
csab
```

含义：修改到下一个 `ab`。

## 单字符和重复跳转

### 接近 `f` 的用法

```text
sa<Enter>
```

含义：跳到最近的 `a`。

### 重复上次跳转

```text
s<Enter>
```

继续按 `<Enter>` 会前进到下一个目标。

### 回退一个目标

```text
<Backspace>
```

在方向跳转过程中可以往回退。

## 特殊目标

### 跳到行尾

```text
sa<Space>
```

含义：匹配 `a` 后面的行尾。

### 跳到空行

```text
s<Space><Space>
```

含义：跳到空行位置。

## 需要注意的地方

- `s` 不再是 Vim 原生的替换单字符，原生行为可用 `cl`
- `S` 不再是整行替换，原生行为可用 `cc`
- Visual 模式下的 `x` 不再是默认删除，删除仍可直接用 `d`

## 记忆版

```text
s   向前找
S   向后找
gs  跨窗口找
d/y/c + s + 两字符 = 对目标执行操作
```
