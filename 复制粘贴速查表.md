# Neovim 复制/粘贴速查表

这份速查表基于你当前这套 `nvim` 配置。

补充说明：

- `<leader>` 是空格键
- 你的 `y / p / P` 已经由 `yanky.nvim` 接管，但基本用法和 Vim 原生一致
- `"<寄存器>` 表示显式指定寄存器
- 常见系统剪贴板寄存器：
  - `"+`：系统剪贴板
  - `"*`：主选择区剪贴板
  - `"`：默认寄存器
  - `"0`：最近一次 yank 的内容
- 本地非 SSH 时，普通复制通常会和系统剪贴板同步
- SSH 场景下不保证自动同步系统剪贴板，此时更建议显式用 `"+`

## 普通模式

### 复制

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 复制当前行 | `yy` | 最常用 |
| 复制 3 行 | `3yy` | 从当前行开始复制 3 行 |
| 复制一个单词 | `yiw` | yank inner word |
| 复制括号内内容 | `yi(` | 同类还有 `yi[`、`yi{`、`yi"` |
| 复制到行尾 | `y$` | 从光标处复制到行尾 |
| 复制到行首 | `y0` | 从光标处复制到行首 |
| 复制到某个字符前 | `yt,` | 复制到 `,` 之前 |
| 复制整个文件 | `ggVGy` | 先全选再复制 |
| 复制到系统剪贴板 | `"+yy` | 显式写入系统剪贴板 |
| 复制一个单词到系统剪贴板 | `"+yiw` | 适合复制变量名、函数名 |

### 粘贴

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 在光标后粘贴 | `p` | 最常用 |
| 在光标前粘贴 | `P` | 粘贴到前面 |
| 粘贴后光标跟到新文本后面 | `gp` | 适合连续编辑 |
| 粘贴前光标停在新文本前面 | `gP` | 便于继续观察前面内容 |
| 从系统剪贴板粘贴 | `"+p` | 显式从系统剪贴板粘贴 |
| 从最近一次 yank 粘贴 | `"0p` | 不受删除寄存器影响 |

### Yank 历史

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 打开复制历史 | `<leader>p` | 查看最近复制过的内容 |
| 切到上一条复制记录 | `[y` | 在 yank 历史里回退 |
| 切到下一条复制记录 | `]y` | 在 yank 历史里前进 |

## 可视模式

### 复制

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 选中后复制 | `v` 选中后按 `y` | 字符级复制 |
| 行选中后复制 | `V` 选中后按 `y` | 行级复制 |
| 块选中后复制 | `<C-v>` 选中后按 `y` | 列块复制 |
| 复制到系统剪贴板 | 选中后 `"+y` | 明确写入系统剪贴板 |

### 粘贴

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 用寄存器内容替换选区 | 选中后按 `p` | 最常用 |
| 在替换后让光标更自然地停留 | 选中后按 `gp` | yanky 的增强粘贴 |
| 用系统剪贴板替换选区 | 选中后 `"+p` | 明确从系统剪贴板粘贴 |

## 插入模式

插入模式下通常不直接使用 `y` 复制，而是：

- 先按 `<Esc>` 回普通模式再复制
- 或者在插入模式里直接插入某个寄存器内容

### 粘贴

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 插入默认寄存器内容 | `<C-r>"` | 最基础的插入模式粘贴 |
| 插入系统剪贴板内容 | `<C-r>+` | 最推荐记住的一个 |
| 插入最近一次 yank 内容 | `<C-r>0` | 不容易被删除操作污染 |
| 插入主选择区内容 | `<C-r>*` | 有些终端环境会用到 |

### 实用建议

- 需要复制时，先按 `<Esc>` 回普通模式
- 需要从系统剪贴板粘贴时，优先记 `<C-r>+`

## 终端模式

终端模式和普通编辑模式不一样。

在终端模式里，按键默认直接发给 shell，所以：

- 不能像普通模式那样直接用 `y / p` 做文本编辑
- 更常见的是用终端模拟器本身的复制/粘贴快捷键

### 从终端模式切回可编辑状态

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 退出终端输入，进入 Terminal-Normal | `<C-\><C-n>` | 这是最重要的一个 |
| 双击 `<Esc>` 隐藏 Snacks 终端 | `<Esc><Esc>` | 你当前配置里的终端隐藏快捷键 |

### 复制终端输出

有两种常见方法。

#### 方法 1：用终端模拟器快捷键

| 操作 | 常见快捷键 | 说明 |
| --- | --- | --- |
| 复制 | `Ctrl-Shift-C` | 依赖你的终端软件 |
| 粘贴 | `Ctrl-Shift-V` | 依赖你的终端软件 |

这是终端里最稳的方案。

#### 方法 2：进入 Terminal-Normal 后按 Vim 方式操作

1. 按 `<C-\><C-n>`
2. 用 `v`、`V` 或 `<C-v>` 选中内容
3. 按 `y` 复制，或按 `"+y` 复制到系统剪贴板

### 向终端里粘贴

| 操作 | 快捷键 | 说明 |
| --- | --- | --- |
| 用终端模拟器粘贴 | `Ctrl-Shift-V` | 最稳 |
| 某些终端可用主选择区粘贴 | 中键 | 依终端而定 |

不建议在终端模式里把普通模式的 `p` 当作 shell 粘贴来用，因为这不是同一个语义。

## 最值得记住的 10 个

如果只记最实用的，记下面这些就够了：

1. `yy`：复制当前行
2. `yiw`：复制当前单词
3. `p`：在后面粘贴
4. `P`：在前面粘贴
5. `"+yy`：复制到系统剪贴板
6. `"+p`：从系统剪贴板粘贴
7. `<leader>p`：打开复制历史
8. `[y` / `]y`：切换复制历史
9. `<C-r>+`：插入模式粘贴系统剪贴板
10. `<C-\><C-n>`：终端模式退出到 Terminal-Normal

## 一句话建议

- 写代码正文：优先用普通模式的 `y / p / P`
- 和系统互通：优先用 `"+y` / `"+p`
- 插入模式粘贴：优先用 `<C-r>+`
- 终端窗口里操作：优先用终端模拟器快捷键，或者先 `<C-\><C-n>`
